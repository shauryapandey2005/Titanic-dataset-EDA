# Titanic-dataset-EDA
A comprehensive implementation of Logistic Regression to predict passenger survival on the Titanic. Includes detailed Exploratory Data Analysis (EDA) and data preprocessing using Python, Pandas, and Seaborn

##Titanic Survival Prediction using Logistic Regression

This repository contains a comprehensive Exploratory Data Analysis (EDA) and a Logistic Regression model to predict passenger survival on the Titanic.
Using the famous Titanic dataset from Kaggle, the project demonstrates a complete machine learning workflow from data cleaning to model evaluation.
